### 3. 定量分析: 分部估值法——揭示双速增长下的真实价值

#### 3.1 估值方法论

鉴于电工合金各业务板块在成长阶段、盈利能力、资本强度和风险暴露方面存在巨大差异，采用单一的整体估值方法（如P/E或EV/EBITDA）将无法公允地反映其内在价值。例如，将高增长、高毛利的新能源汽车业务与成熟、低毛利的铜母线业务用同一个估值倍数衡量，会严重扭曲公司的真实价值。

因此，我们坚决采用**分部估值加总法（Sum-of-the-Parts, SOTP）**作为本次分析的核心定量工具。该方法允许我们为每个业务板块选择最合适的估值指标（如EV/Sales）和估值倍数，从而更精确地捕捉其独特的价值贡献。我们将基于各分部的远期（2025财年预测）收入，并结合其增长前景、竞争格局和盈利能力，赋予其合理的估值倍数，最终汇总得出公司的整体企业价值（EV）。

#### 3.2 估值过程详解

我们的估值基准日为2025年9月16日，所有财务数据均基于最新的公开披露信息，包括2024年年度报告[2]和2025年半年度报告[1]。

### 4. 定性分析: 护城河与阿喀琉斯之踵——增长叙事背后的风险矩阵

定量估值给出了价值的“锚”，而定性分析则解释了这艘船的航行能力与潜在风险。电工合金的投资故事，正是一场关于其坚固护城河与致命弱点之间的博弈。

**护城河：技术、认证与客户粘性**

公司的核心竞争力并非源于规模或成本，而是建立在难以复制的软实力之上。
*   **技术与认证壁垒（高强度）：** 在电气化铁路领域，产品质量直接关系到国家运输大动脉的安全。电工合金的产品不仅需要通过中国铁路集团等机构的严苛认证，形成了极高的准入门槛，更在技术上不断突破。公司在2025年半年报中披露，已初步完成适用于时速400公里以上的高强度铜合金接触线研发，其抗拉强度等关键指标已达到国际领先水平[1]。这种技术代差是其获取高价值订单（尤其是海外定制化项目）的关键。
*   **大客户粘性（中高强度）：** 在输配电领域，公司与施耐德、ABB、西门子等全球巨头建立了长期的供应商关系。这些客户对供应商的质量控制、交付能力和研发协同要求极高，一旦认证通过，更换成本巨大。这种深度绑定关系为公司的传统业务提供了宝贵的稳定性。

**阿喀琉斯之踵：脆弱的现金循环**

然而，再宽的护城河也无法掩盖公司运营层面的核心缺陷——孱弱的现金流管理能力。
*   **高企的应收账款：** 公司最新的财务数据显示，其应收账款周转天数（DSO）高达121天[5]。这意味着产品售出后，平均需要4个月才能收回现金。这一方面反映了其在产业链中对下游大客户（如大型工程承包商）议价能力的不足，另一方面也极大地占用了公司的运营资金。截至2025年6月30日，公司账面净应收账款高达9.27亿人民币，几乎是同期净资产的80%[3]。
*   **负向的自由现金流：** 高应收和高库存（库存周转天数约67天[5]）共同导致了长达161天的现金循环周期（CCC）[5]。其直接后果是，公司在过去12个月（TTM）的自由现金流为负。这意味着公司依靠主营业务赚取的现金不足以覆盖其资本开支和日常运营，需要持续依赖外部融资（如银行借款）来“输血”。这种“增收不增利，增利不增金”的模式是不可持续的，也是我们对其高估值最为警惕的核心原因。
*   **对铜价的敏感性：** 公司的成本结构中，原材料（电解铜）占比较高。对于毛利率本就微薄的铜母线业务而言，铜价的剧烈波动会直接侵蚀其盈利空间。虽然公司可以通过部分价格传导和套期保值来对冲风险，但高库存策略使其在铜价下行周期中面临存货跌价的风险，在铜价上行周期中又面临营运资金需求激增的压力。
